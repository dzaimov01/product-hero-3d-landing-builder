# Design System

## Type scale
- H1: 36-48px, display serif.
- H2: 28-32px, sans.
- Body: 15-17px, sans.
- Caption: 11-12px, uppercase tracking.

## Spacing
Use the spacing tokens defined in `src/styles/themeTokens.ts` and `src/styles/tokens.css`.

## Radius & shadow
- Radius: 0.5rem, 0.875rem, 1.5rem.
- Shadow: soft ambient `0 20px 50px rgba(6, 10, 18, 0.2)`.

## Color tokens
- Ink (surface): #0a0c10 - #242f3f
- Linen (light): #f7f5f2
- Aurum (accent): #d6b36a
